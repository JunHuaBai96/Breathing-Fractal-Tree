## ✨ 效果预览

<img width="1076" height="705" alt="image" src="https://github.com/user-attachments/assets/bf2f04a4-4388-4804-b416-7253ff852544" />



| 特性 | 描述 |
|:---|:---|
| 🌿 **动态摇曳** | 树枝角度随 `sin(millis())` 周期性变化，模拟自然风动 |
| 🎨 **色彩渐变** | 树干棕色平滑过渡至末梢嫩绿 |
| 📐 **递归分形** | 深度上限控制，越往末端越细越密 |
| 🍂 **交互落叶** | 点击屏幕触发柔和叶片洒落动效 |

## 🚀 快速开始

无需安装，浏览器即开即用：

```bash
# 克隆仓库
git clone https://github.com/JunHuaBai96/breathing-fractal-tree.git

# 进入目录
cd breathing-fractal-tree

# 用浏览器打开
open index.html
